### 环境依赖

node v4.5.0 以上

### 安装

```
$ npm install mdev-pre -g
```

### 更新资源包

```
$ mdev-pre update
```

### 查看版本

```
$ mdev-pre -v
```

### 编译代码

* 进入目录

```
$ cd project-dir
```

* 执行编译

```
$ mdev-pre build
```

* 执行编译并监听文件变化

```
$ mdev-pre build -w
```

* 启动server

```
$ mdev-pre start
```
