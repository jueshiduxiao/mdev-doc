配置文件：src/config.json

说明：

* mdev-template 框架模板类型 当前框架值为：vue
* public-path 编译代码路径
* debug 开发模式 \[true\|false\]
* data-model 数据模式
* global-library 公共依赖资源

```json
{
    "mdev-template": "vue",
    "public-path": "../dist/",
    "debug": false,
    "data-model": "full",
    "element-ui": true,
    "global-library": [
        "/src/ui/lib/reset.v3.1.1.css",
        "/src/ui/lib/global.css"
    ],
    "dependencies": {
    }
}
```
